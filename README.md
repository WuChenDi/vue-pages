# 🚀 vue-pages

`vue2.6 + ts` 示例，补充一些扩展功能

- [x] 多页面应用
- [x] 多环境隔离
- [ ] 多语言 loader 开发

## Project setup

```sh
npm install
```

### Compiles and hot-reloads for development

```sh
npm run serve
```

### Compiles and minifies for production

```sh
npm run build

cd ./dist/master

npm i -g http-server

http-server -c-1
```

### Run your tests

```sh
npm run test
```

### Lints and fixes files

```sh
npm run lint
```
